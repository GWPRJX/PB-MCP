POSibolt API patterns (all POST to /AdempiereService/PosiboltRest/...):
- Stock transfer request: POST /stocktransferrequest — body: { dateFormat, dateRequired, fromWarehouseId, toWarehouseId, lines[] }
- Complete stock transfer: POST /stocktransfer/completestocktransfer — body: { comments, issuedBy, toWarehouseId, transferDate, stockTransferId, lines[] }
- Create invoice: POST /salesinvoice/createorderinvoice — body: { dateInvoiced, invoiceNo, dateFormat, customerId, priceListId, paymentType, grandTotal, warehouseId, invoiceLineList[], payments[] }
- Cancel order: POST /salesorder/cancelorder — body: { orderNo }
- Create customer: POST /customermaster — body: { customerCode, name, action:"create", active, region, address1, ... }
- Update customer: POST /customermaster/{customerId} — body: { customerId, name, action:"update", active, ... }

POSibolt success response pattern:
```json
{ "responseCode": 200, "message": "Success", "record": "customer", "recordNo": 10000001, "detailedMessage": "Entry Created Successfully" }
```

MCP tool registration pattern (from existing tools):
```typescript
export function registerWriteTools(server: McpServer, filter?: Set<string> | null): void {
  if (shouldRegister('tool_name', filter)) server.tool(
    'tool_name',
    'Tool description',
    { /* zod schema */ },
    withAudit('tool_name', async (params) => {
      try {
        const config = getErpConfig();
        const result = await pbPost(config, '/endpoint', body);
        return toolSuccess(result);
      } catch (err) {
        process.stderr.write(`[tools/write] tool_name error: ${err instanceof Error ? err.message : String(err)}\n`);
        return toolError('INTERNAL_ERROR', err instanceof Error ? err.message : 'Unknown error');
      }
    }),
  );
}
```

    - `lines` — z.array(z.object({...})).min(1) — Line items (required):
      - `productId` — z.number().int() — Product ID (required)
      - `qty` — z.number().positive() — Quantity to transfer (required)
      - `uom` — z.string().default('Each') — Unit of measure
      - `sku` — z.string().optional() — Product SKU
      - `productName` — z.string().optional() — Product name
  - Handler:
    - Call `getErpConfig()`
    - Format date using `toPosiDate()` (default to today if not provided)
    - Build request body:
      ```typescript
      {
        dateFormat: 'dd-MM-yyyy',
        dateRequired: formattedDate,
        fromWarehouseId: params.fromWarehouseId,
        toWarehouseId: params.toWarehouseId,
        lines: params.lines.map(l => ({
          moveAllQty: false,
          productId: l.productId,
          qty: l.qty,
          reqQty: l.qty,
          uom: l.uom ?? 'Each',
          sku: l.sku ?? '',
          productName: l.productName ?? '',
          upc: '',
          warehouseId: 0,
        })),
      }
      ```
    - POST to `/stocktransferrequest`
    - Return `toolSuccess(result)`
