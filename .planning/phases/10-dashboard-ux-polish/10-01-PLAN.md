const maskKey = (key: string) => {

Note: API keys start with `pb_` so masking like `pb_a1b****ef56` is the expected output. Since we don't have the raw key for existing keys (only hash stored), we need to show a notice: "Full API key is only available when first created. Use a key label to identify which key to use."
