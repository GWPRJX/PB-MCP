  <name>Task 1: Add KB doc CRUD endpoints to admin router</name>
  <files>src/admin/router.ts</files>
  <action>
  Add 5 new endpoints to `src/admin/router.ts` inside the `adminRouter` function, after the existing `GET /admin/tools` endpoint. All endpoints are automatically protected by the existing `jwtAuthHook`.

  Add these imports at the top of router.ts (neither currently exists in the file):
  ```typescript
  import { createHash } from 'crypto';
  import { sql } from '../db/client.js';
  ```

  **1. POST /admin/kb/upload** — Upload a new API doc
  ```
  Body: { title: string, content: string, tags?: string[] }
  ```
  - Generate a unique youtrack_id: `DOC-${crypto.randomUUID().split('-')[0]}` (8-char hex prefix)
  - Compute content_hash: `createHash('sha256').update(content).digest('hex')`
  - Validate: title required (min 1 char), content required (min 1 char), content max 1MB (1_048_576 chars)
  - Insert into kb_articles: `INSERT INTO kb_articles (youtrack_id, summary, content, tags, content_hash) VALUES (...) RETURNING id, youtrack_id, summary, tags, synced_at`
  - Return 201 with the created doc object
  - Schema: body { title: string, content: string, tags?: string[] }, response 201 { id, youtrackId, title, tags, createdAt }

  **2. GET /admin/kb/docs** — List uploaded docs
  ```
  Querystring: { limit?: string, offset?: string }
  ```
  - Query: `SELECT id, youtrack_id, summary, tags, synced_at FROM kb_articles WHERE youtrack_id LIKE 'DOC-%' ORDER BY synced_at DESC LIMIT $limit OFFSET $offset`
  - Also get total count: `SELECT COUNT(*) FROM kb_articles WHERE youtrack_id LIKE 'DOC-%'`
  - Return: `{ docs: [...], totalCount: number }`
  - Map column names for frontend: youtrack_id → youtrackId, summary → title, synced_at → createdAt

  **3. PUT /admin/kb/docs/:id** — Update an uploaded doc
  ```
  Params: { id: string } (the uuid primary key)
  Body: { title?: string, content?: string, tags?: string[] }
  ```
  - Only update fields that are provided (non-undefined)
  - If content is updated, recompute content_hash
  - Update synced_at to NOW()
  - If title is provided, update summary
  - Query: Build dynamic UPDATE using provided fields
  - Verify the doc exists AND has `youtrack_id LIKE 'DOC-%'` (prevent editing YouTrack-synced articles)
  - Return 200 with `{ updated: true }` or 404 with `{ error: "Doc not found" }`

  **4. DELETE /admin/kb/docs/:id** — Delete an uploaded doc
  ```
  Params: { id: string }
  ```
  - Delete: `DELETE FROM kb_articles WHERE id = $id AND youtrack_id LIKE 'DOC-%'`
  - Check rowCount — if 0, return 404 `{ error: "Doc not found" }`
  - Return 204 on success
  - Only allows deleting uploaded docs (DOC-* prefix guard)

  **5. GET /admin/kb/docs/:id** — Get single uploaded doc with full content
  ```
  Params: { id: string } (the uuid primary key)
  ```
  - Query: `SELECT id, youtrack_id, summary, content, tags, synced_at FROM kb_articles WHERE id = $id AND youtrack_id LIKE 'DOC-%'`
  - Map column names for frontend: youtrack_id → youtrackId, summary → title, synced_at → createdAt
  - Return 200 with full doc object (including content), or 404 `{ error: "Doc not found" }`
  - This endpoint is used by the dashboard to load full content for editing (the list endpoint omits content to keep responses light)

  **1. dashboard/src/api.ts** — Add types and API functions for doc management:

  a) Add interface:
  ```typescript
  export interface KbDoc {
    id: string;
    youtrackId: string;
    title: string;
    tags: string[];
    createdAt: string;
  }
  ```

  b) Add `KbDocFull` interface (extends KbDoc with content):
  ```typescript
  export interface KbDocFull extends KbDoc {
    content: string;
  }
  ```

  c) Add API functions:
  ```typescript
  export const uploadDoc = (data: { title: string; content: string; tags?: string[] }) =>
    api<KbDoc>('/kb/upload', { method: 'POST', body: JSON.stringify(data) });

  export const listDocs = (params?: { limit?: number; offset?: number }) => {
    const qs = new URLSearchParams();
    if (params?.limit) qs.set('limit', String(params.limit));
    if (params?.offset) qs.set('offset', String(params.offset));
    const q = qs.toString();
    return api<{ docs: KbDoc[]; totalCount: number }>(`/kb/docs${q ? `?${q}` : ''}`);
  };

  export const getDoc = (id: string) =>
    api<KbDocFull>(`/kb/docs/${id}`);

  export const updateDoc = (id: string, data: { title?: string; content?: string; tags?: string[] }) =>
    api<{ updated: boolean }>(`/kb/docs/${id}`, { method: 'PUT', body: JSON.stringify(data) });

  export const deleteDoc = (id: string) =>
    api<void>(`/kb/docs/${id}`, { method: 'DELETE' });
  ```
