### TC-8.2: Sync Now Button

## Phase 9: Tenant Onboarding Flow

### TC-9.1: Wizard Visual Flow
- [ ] Navigate to `/tenants/new`
- [ ] **Expected:** Horizontal stepper shows 3 labeled steps ("Tenant Info", "ERP Credentials", "API Key"). Step circles show correct completed/current/future visual states as you progress.

### TC-9.2: Test Connection Feedback
- [ ] On Step 2, enter invalid ERP credentials and click "Test Connection"
- [ ] **Expected:** Red error box appears with a helpful hint specific to the failure type (DNS, auth, timeout). "Next" button remains disabled.

### TC-9.3: Test Connection Success Gate
- [ ] On Step 2, enter valid ERP credentials and click "Test Connection"
- [ ] **Expected:** Green success indicator. "Next" button becomes enabled. Cannot skip to Step 3 without passing test.

### TC-9.4: API Key Copy
- [ ] Complete tenant creation (all 3 steps), click "Copy" on the API key display
- [ ] **Expected:** API key copied to clipboard; button text changes to "Copied!" for ~2 seconds then reverts.
