# Sales Order
These APIs are used to post or cancel sales orders. This just creates the order only, and it will not generate the invoice. invoice will have to be created separately eithe
through API or through Posibolt.

✔ Mandatory
-- Optional

| Action | URL | Method | Parameters |
| --- | --- | --- | --- |
| Post a Sale Order | `/salesorder/createorder` | POST | JSON Body ✔ |
| Cancel a Sale Order | `/salesorder/cancelorder` | POST | JSON Body ✔ |
| Fetch Customer Order History | `/salesorder/customerhistory/{customerId}` | GET | **✔ customerId**, `offset` –, `limit` –, `fromDate` -- |
| Fetch All Invoices of Order | `/salesorder/getAllInvoices` | GET | `orderNo` ✔ |
| Fetch Status of Order | `/salesorder/orderstatus` | GET | `orderNo`, `updatedSince`, `customerId`, `customerCode` (At least one of the parameters is mandatory) |
