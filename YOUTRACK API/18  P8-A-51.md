# Product Discount Prices
* **URL :**  /productmaster/getDiscountedPrices
* **Method**: GET
* **Description :** The Api retrieves  discounted prices for the products.
* **Query Parameters**:
  1. `updatedSince` **(optional):** Return products updated on or after the given timestamp (milliseconds since epoch).
  2. `productIds` **(optional):** Comma-separated product list IDs to filter product prices. Leave blank to include all.
  3. `offset` **(optional):** Number of records to skip for pagination. Default is 0.
  4. `limit` **(optional):** Maximum number of records to return. Default is 0 (no limit).

NB:

**Product discount prices are calculated based on the sales price list.**



**Sample Request:**

[http://localhost:8088/AdempiereService/PosiboltRest/productmaster/getDiscountedPrices](http://127.0.0.1:8088/AdempiereService/PosiboltRest/productmaster/getDiscountedPrices)
