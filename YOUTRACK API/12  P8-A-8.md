# Customer Administration
This is the API to manage customers . Every POST method written below should have request body like the format given below:

* ✔ Mandatory
* -- Optional


| Action                      | URL                                            | Method | Parameters                                                                            |
| --------------------------- | ---------------------------------------------- | ------ | ------------------------------------------------------------------------------------- |
| **Create Customer**         | `/customermaster`                              | POST   | JSON Body ✔                                                                           |
| **Update Customer**         | `/customermaster/{customerId}`                 | POST   | **customerId** ✔, JSON Body ✔                                                         |
| **Retrieve Customer**       | `/customermaster/{customerId}`                 | GET    | **customerId**   ✔                                                                   |
| **Retrieve Customer List**  | `/customermaster/customerslist`                | GET    | `updatedSince` --, `allLoc` – (default=false), `offset` –, `limit` –, `bpGroupName` -- |
| **Create Customer Address** | `/customermaster/customeraddress/{customerId}` | POST   | **customerId** ✔, JSON Body ✔                                                         |
| **Fetch All Addresses**     | `/customermaster/getalladdress`                | GET    | **customerId** ✔                                                                      |
| **Search Customers**        | `/customermaster/searchcustomers`              | GET    | `customerCode` – , `email` –, `mobileNo` –, `name` --                                           |


**JSON**
1. Create/Update Customer
`{`
`  "customerId": 1000001,`
`  "customerCode": "CUST00321",`
`  "name": "testname",`
`  "customerGroup": "standard",`
`  "address1": "testaddress1",`
`  "address2": "testaddress2",`
`  "city": "testcity",`
`  "country": "india",`
`  "region": "testState",`
`  "email": "testmail@gmail.com",`
`  "active": true,`
`  "action": "create"`
`}`

| Field          | Description                                               | Mandatory  | Type    |
| -------------- | --------------------------------------------------------- | ---------- | ------- |
| customerId     | Internal ID in DB. Needed for updates.                    | ✔ (update) | Integer |
| customerCode   | Unique code (e.g., medical record no)                     | ✔          | String  |
| name           | First name of the customer                                | ✔          | String  |
| region         | Region/State                                              | ✔          | String  |
| action         | `"create"`, `"update"`, `"delete"`                        | ✔          | String  |
| active         | Active or Inactive                                        | ✔          | Boolean |
| customerGroup  | Name of customer group. Default applied if not given.     | –          | String  |
| address1       | Address line 1                                            | ✔          | String  |
| address2-4     | Additional address lines                                  | –          | String  |
| city           | City                                                      | –          | String  |
| country        | Country                                                   | –          | String  |
| mobile / phone | Contact numbers                                           | –          | String  |
| postalCode     | Postal/Zip Code                                           | –          | String  |
| email          | Email address                                             | –          | String  |
| fax            | Fax number                                                | –          | String  |
| creditStatus   | Credit status (Credit Ok / No Credit Check / Credit Stop) | –          | String  |
| creditLimit    | Allowed outstanding amount                                | –          | Numeric |
| pricelistId    | Price list ID                                             | –          | Integer |

**Success Response Example**

`{`
`  "responseCode": 200,`
`  "message": "Success",`
`  "record": "customer",`
`  "recordNo": 10000001,`
`  "detailedMessage": "Entry Created Successfully"`
`}`

**Failure Response Example**

`{`
`  "responseCode": 400,`
`  "message": "Bad Request",`
`  "detailedMessage": "Customer name already exists"`
`}`
