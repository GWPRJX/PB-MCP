# Product Search - Uom Price Handling
* **URL:** /productmaster/search
* **Method:** GET

Added new fields in the `Product DTO` and `UoM DTO` for UoM Conversion products

* **convPriceDetails -** Get the list of uom conversions (Product DTO)
* **uomStdPrice -** Get the uom standard price of the product (UoM DTO)
