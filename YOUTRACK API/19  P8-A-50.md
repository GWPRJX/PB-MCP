# Detailed Product Information
* **URL : /productmaster/productDetailedList**
* **Method**: GET
* **Description :** The API retrieves detailed product information, including pricing, stock availability, categorization, and warehouse-wise stock quantities, enabling seamless synchronization between ERP/POS systems and third-party applications.
* **Query Parameters**:
   1. `updatedSince` **(required):** Return products updated on or after the given timestamp (milliseconds since epoch).
   2. `includeStockUpdate` **(optional):** Include products with stock changes since the given date. Default is true.
   3. `priceListIds` **(optional):** A comma-separated list of Price List IDs used to filter product prices. If not provided, the default Price List will be used for filtering.
   4. `webstoreFeaturedOnly` **(optional):** Return only products marked as webstore featured. Default is false.
   5. `offset` **(optional):** Number of records to skip for pagination. Default is 0.
   6. `limitByOrg` **(optional):** Restrict product results to the logged-in organization. Default is false.
   7. `isactive` **(optional):** Return only active products if true. Default is false.
   8. `limit` **(optional):** Maximum number of records to return. Default is 0 (no limit).
   9. `skuType` **(optional):** Filter products by SKU type. Leave blank to include all types.
  10. `excludeInTransit` **(optional):** Exclude stock quantities that are in transit. Default is false.
  11. `sortByName` **(optional):** Sort products alphabetically by name if true. Default is true.
  12. `baseProductId` **(optional):** Return only products belonging to the given base product ID. Default is 0.

  
  

NOTE:

Parameter - pricelistIds

* If pricelistIds are passed, the specified price lists are checked, and the products whose prices were updated in those price lists will be returned.
* If pricelistIds are not passed, the default price list is considered. In that case, only the products whose prices were updated in the default price list will be returned.
