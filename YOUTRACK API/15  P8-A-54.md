# Fetch Product Prices
* **URL** :`/productmaster/getproductprices`
* **Method**: GET
* **Description** : The Api retrieves  prices for the products.
* **Query Parameters**:

  1.`productIds`**(Required) -** Comma-separated product list IDs to filter product prices.

  2.`priceListId`(**Optional**) - Return only products belonging to the given pricelist Id. Default 0.

  3.`updatedSince`(**Optional**) - Return products updated on or after the given timestamp (milliseconds since epoch).Default is 0.




NOTE:

* If priceListId is passed and the FETCH_PRODUCT_PRICE_FOR_SINGLE_PL sysconfig is set to true, only that specific Price List will be considered, and only the products with updated prices in that Price List will be returned.
* If the `priceListId` is passed but the config is set to **false**, only the Active Price Lists that are marked as available for products (`isPresentForProduct` and `isActive`) will be considered, and the products with updated prices in those Price Lists will be displayed.
* If no priceListId is passed, only Active Price Lists that are marked as available for products (isPresentForProduct and isActive ) will be considered, and products with updated prices in those Price Lists will be displayed..




**Sample Request:**

<http://127.0.0.1:8088/AdempiereService/PosiboltRest/productmaster/getproductprices?productIds=1617738,1594804&priceListId=1000678>
