# Canceling the Order
This API cancels a sales order. Cancellation is allowed only for orders that have not been invoiced yet. To cancel an invoiced order, a Sales Return must be performed.

**Endpoint**

| Action              | URL                       | Method | Parameters  |
| ------------------- | ------------------------- | ------ | ----------- |
| Cancel a Sale Order | `/salesorder/cancelorder` | POST   | JSON Body ✔ |


**JSON:**

**Example Request:**

`{`
`    "orderNo": "HISORD005",`
`    "customerCode": "MRN001",`
`    "description": "some comments",`
`    "dateOrdered": "09-03-2015",`
`    "warehouseId": 15423,`
`    "orderLineList": [`
`        {`
`            "productId": 1009524,`
`            "qty": 1,`
`            "price": 10,`
`            "uom": "TAB"`
`        },`
`        {`
`            "productId": 1009525,`
`            "qty": 1,`
`            "price": 10,`
`            "uom": "TAB"`
`        }`
`    ]`
`}`

**Example Response (Success)**

`{`
`    "responseCode": 200,`
`    "detailedMessage": "Entry Cancelled Successfully",`
`    "record": "Order",`
`    "recordNo": "C1",`
`    "message": "Sucess"`
`}`

**Example Response (Failure)**

`{`
`  "responseCode": 600,`
`  "detailedMessage": "Order number not found",`
`  "record": null,`
`  "recordNo": null,`
`  "message": "Bad Request"`
`}`

**Field Description**

| Field         | Description                      | Mandatory | Type    |
| ------------- | -------------------------------- | --------- | ------- |
| orderNo       | Unique ID of the order to cancel | ✔         | Numeric |
| customerCode  | Unique code of the customer like medical record no:                 | –         | String  |
| description   | Any comments on the sale order                 | –         | String  |
| dateOrdered   | Date when the cancel order is created                   | ✔         | String  |
| salesRepId   | Unique Id of the user                  | --         | String  |
| warehouseId   | Warehouse ID for stock dispatch  | ✔         | Numeric |
| orderLineList | List of products to cancel       | ✔         | Array   |


**Order Line fields:**

| Field     | Description        | Mandatory | Type    |
| --------- | ------------------ | --------- | ------- |
| productId | Product ID         | ✔         | Numeric |
| qty       | Qty of the product ordered | ✔         | Numeric |
| price     | Unit price         | --         | Numeric |
| uom       | Unit of measure    | --        | String  |
