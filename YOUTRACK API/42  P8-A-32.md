# Warehouse/Stock API
**Warehouse Master API Endpoints**

✔ Mandatory
**--** Optional

| Endpoint | Method | Purpose | Required Parameters / Notes |
|-----------|---------|----------|-----------------------------|
| `/warehousemaster/warehouselist` | **GET** | Get list of all warehouses (for user’s org(s)) | `adOrgId` *(**--**, 0 = all orgs)* |
| `/warehousemaster/mywarehouselist` | **GET** | Get warehouse list for current org | `editable` *(**--**, true = only editable warehouses, false = all viewable)* |
| `/warehousemaster/{warehouseId}` | **GET** | Get details of a specific warehouse | `warehouseId` ✔ |
| `/warehousemaster/default` | **GET** | Get default warehouse for an org | `adOrgId` *(**--**, 0 = user’s org)* |
| `/warehousemaster/current` | **GET** | Get current (logged-in) warehouse (contextual for user/terminal) | — |
| `/warehousemaster/locator/{locatorId}` | **GET** | Get locator (location within warehouse) details | `locatorId` ✔ |
| `/warehousemaster/locatorlist` | **GET** | Get all locators for a warehouse | `warehouseId` ✔ |
| `/warehousemaster/locator/default` | **GET** | Get default locator for a warehouse | `warehouseId` ✔ |
| `/warehousemaster/getWareHouseInventory` | **GET** | Get storage/stock list for a warehouse | `warehouseId` **--**, `groupByLocator` *(**--**, default: false)* |
