# Sales Order - Field Description (common across orders)
| Field         | Description                     | Mandatory | Type    |
| ------------- | ------------------------------- | --------- | ------- |
| orderNo       | Unique order number             | --         | String  |
| customerCode  | Unique code of the customer                   | –         | String  |
| description  | Any comments on the sale order                | –         | String  |
| dateOrdered   | Date of order                   | ✔         | String  |
| salesRepId   | Unique Id of the user                   | --         | String  |
| warehouseId   | Id of the warehouse from where the stock will be dispatched/picked up.                  | –         | Numeric |
| invoiceRule   | Rule on how the invoice should have been generated. Possible Values are : "Customer Schedule after Delivery" (Default) , "After Delivery"     | –         | String  |
| paymentType   | Payment mode (Card, Cash, etc.) | --         | String  |
| grandTotal   | Order Total |   ✔       | Numeric  |
| freightAmt   | Freight Charge Amount |   --       | Numeric  |
| comment   | Special comment to mention in the order | --         | String  |
| orderLineList | List of products ordered        | ✔         | Array   |
| payments      | List of payments made           | –         | Array   |



**Order Line Fields:**

| Field     | Description      | Mandatory | Type    |
| --------- | ---------------- | --------- | ------- |
| productId | Id of the product ordered. This could be Id of the item or a service.       | ✔         | Numeric |
| qty       | Quantity ordered | ✔         | Numeric |
| price     | Unit price       | ✔         | Numeric |
| uom       | Unit of measure  | --         | String  |




**Payment Fields:**

| Field                | Description                   | Mandatory | Type    |
| -------------------- | ----------------------------- | --------- | ------- |
| paymentNo            | Unique payment number         | –         | String  |
| amount               | Payment amount                | ✔         | Numeric |
| paymentType          | Payment method                | ✔         | String  |
| cardNo / referenceNo | Card or transaction reference | –         | String  |
