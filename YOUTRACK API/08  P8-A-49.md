# Completed Stock Transfer History
* **URL**: `/stocktransfer/getcompletedsthistory`
* **Method**: GET
* **Description**: Fetch Completed Stock Transfer History
* **Query Parameters**: 

    1. `toWarehouseId` (Optional, int): The ID of the target warehouse.
    2. `documentNo` (Optional, String): The ST document number.
    3. `fromDate` (Optional, Long): Start date for querying transactions.
    4. `toDate` (Optional, Long): End date for querying transactions.
    5. `limit` (Optional, int): The maximum number of records to return.
    6. `offset` (Optional, int): The number of records to skip for pagination. Default is 0.

**Note:**

* Either `documentNo` or both `fromDate` and `toDate` must be provided.
* `fromDate` and `toDate` should be in **milliseconds since epoch**.
* If the `limit` is 0 or more than 100, it will **default to 100** to ensure good performance and avoid overloading the system.




**Sample Request:**
1. By DocumentNo:
<http://0.0.0.0:8088/AdempiereService/PosiboltRest/stocktransfer/getcompletedsthistory?documentNo=1001324>

2. By DateRange:
<http://0.0.0.0:8088/AdempiereService/PosiboltRest/stocktransfer/getcompletedsthistory?fromDate=1769919088000&toDate=1771992688000>
