# Fetch Product Additional UOMs
1. **URL :**  /productmaster/productUoms
2. **Method**: GET
3. **Description :** The Api fetching  additional UoMs of the product.
4. **Query Parameters**: 
   1. `productId` **(required):** Displays the additional UOMs for the selected product.
   2. `isactive` **(optional):** defaultValue (true) -Only active UoMs are fetched. To fetch all active and inactive UoMs, set this parameter to **false**.




**Sample Request**

GET

<http://localhost:8088/AdempiereService/PosiboltRest/productmaster/productUoms?&productId=1594870>



NB:

**Fields that match the required fields.**

Uom  =  toUoMName

multiply rate  =  divideRate

barcode  = upc

Price  = uomStdPrice

cost  = uomCostPrice
